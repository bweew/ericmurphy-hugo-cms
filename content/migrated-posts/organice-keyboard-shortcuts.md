\- next prev header  = ctrl up / down

\- toggle open header = tab

\- toggle todo states = alt t

\- edit title = ctrl h

\- edit desc = ctrl d

\- exit edit = alt enter

\- add header = ctrl enter

\- del header = backspace 

\- move header = alt up/down

\- demote/promote header = alt shift left/right
